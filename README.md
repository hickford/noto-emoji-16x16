# Noto emoji 16x16 pixels

[Noto emoji](https://github.com/googlefonts/noto-emoji) resized to 16x16 pixels.

This is useful for devices with a 16x16 pixel screen such as [Yoto player](https://yotoplay.com/) for children.

## See also

* <https://www.yotoicons.com/>
* [MYO Studio](https://chromewebstore.google.com/detail/myo-studio/olloopaapdodegpcacjmgdjaphggfgoj) Chrome extension

## Index

To download, click the icon then use 'download raw file'. Alternatively, right click and use 'save image as'.

For best results, prefer the icon with transparent background (this appears black on the Yoto player screen).
